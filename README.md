## 🛠️ Инструкция по установке (TrollStore)

> [!IMPORTANT]
> Рекомендую использовать **TrollStore**. Это единственный способ запускать эти игры без переподписи каждые 7 дней.

### 1. Как установить TrollStore
Если у вас его еще нет, следуйте этой схеме (зависит от вашей версии iOS):
* **iOS 14 — 15:** Используйте [TrollHelperOTA](https://trollstore.app).
* **iOS 15 — 16:** Используйте [TrollInstallerX](https://github.com/AlfieCG_dev/TrollInstallerX/releases).
* **iOS 17:** Проверьте возможность установки через [официальный гайд](https://ios.cfw.guide/installing-trollstore/).

### 2. Как установить игру
1. Перейдите по ссылке на **MEGA**.
2. Скачайте нужный `.ipa`.
3. Нажмите на файл -> **Поделиться (Share)** -> **TrollStore**.
4. Нажмите **Install**. Поздравляем, игра установлена навсегда!
